Ejercicio 1

Este ejercicio cosiste en crear una función llamada productTable que reciba como parámetro un número N entero positivo que retorna una matriz con N tablas de multiplicar.

El código desarrollado para este ejercicio es el siguiente:

/**
 * Get n multiplication tables.
 * @param num number of multiplication tables.
 * @returns Arrays of Arrays result.
 */
export function productTable(num: number) {
  const result: number[][] = [];
  const array: number[] = [];
  let array2: number[] = [];
  let mult = 1;
  if (num < 1 || !Number.isInteger(num)) {
    return undefined;
  }

  for (let i = 0; i < num; i++) {
    array.push(i + 1);  
  } 

  for(let i = 0; i < num; i++) {     
    for (let j = 0; j < num; j++) {
      array2.push(array[j] * mult);
    }
    result.push(array2);
    array2 = [];
    mult++;
  }
  
  return result;
}

Los primero que hacemos en la función es comprobar si el número pasado por parámetro en un número entero positivo, en caso contrario se retorna undefined. Posteriormente guardamos en un array la primera tabla de multiplicar, es decir, la tabla multiplicada por uno. Es a partir de esa que vamos multiplicando cada uno de sus elementos por un multiplicador que aumenta con cada iteración y lo guardamos en la matriz result, siendo esta la que finalmente se retorna. Obteniendo así todas las tablas de multiplicar correspondiente.

Ejercicio 2

Este ejercicio consiste en la creación de una clase de números racionales con sus respectivos métodos y atributos para crear una serie de operaciones. En la sesión práctica no tuve tiempo de la completa realización del ejercicio.

La parte en cuestión es la siguiente:

/**
 * Interface that the class RationalNum follows.
 * @param num numerator of the rational number.
 * @param den denominator of the rational number.
 * @function print Print the rational number.
 */
interface Rational {
  num: number;
  den: number;
  print(): void;
}
/**
 * Class RationalNum to create a Rational number following the interface bases.
 * @constructor create the rational number with numerator an denominator param.
 * @param num numerator of the rational number.
 * @param den denominator of the rational number.
 */
export class RationalNum implements Rational {
  constructor(public num: number, public den: number) {     
  }

  /**
   * Simplify a rational number.
   * Check if the module's number is 0 with 2, 3 and 5 and if that is true, divide the num and den.
   */
  sim() {
    while ((this.num % 2 == 0 && this.den % 2 == 0) || (this.num % 3 == 0 && this.den % 3 == 0) || (this.num % 5 == 0 && this.den % 5 == 0)) {
      if (this.num % 2 == 0 && this.den % 2 == 0) {
        this.num /= 2;
        this.den /= 2;
      } else if (this.num % 3 == 0 && this.den % 3 == 0) {
        this.num /= 3;
        this.den /= 3;
      } else if (this.num % 5 == 0 && this.den % 5 == 0) {
        this.num /= 5;
        this.den /= 5;
      }
    }

  }
  /**
   * Print the rational number.
   */
  print() {
    console.log(`${this.num} / ${this.den}`);
  }
}

const numb = new RationalNum(2, 4);

numb.print();
numb.sim();
numb.print();

Cómo vemos lo primero es la creación de una interfaz para la posterior clase que nos obligue a tener un numerador, denominador y una función print(). En la clase RationalNum pasamos por paŕametro tanto el numerador como el denominador y la función implementada a parte de print() es sim() la cual nos permite simplificar un número racional. Para esto se comprueba si tanto el numerador como el denominador son divisibles por 2, 3 o 5 y si es así se hacen las divisiones correspondientes.

Las pruebas llevadas a cabo de mocha y chai para comprobar su correcto funcionamiento es el siguiente:

    ✔ (30/12) sim returns 5 / 2

Ejercicio 1 - Biblioteca musical

Este ejercicio consiste en la creación de una biblioteca musical haciendo uso de las clases necesarias, incluyendo tres elementos fundamentales: artista, discografía y canción. Además deberań de crearse una serie de métodos para conseguir hacer ciertas operaciones.

const MusicLibrary: Artist[] = [];

/**
 * Song.
 * @param name Song's name string.
 * @param duration Number of song's duration.
 * @param gender Song's gender gender.
 * @param single Boolean, true if is a single, false if not.
 * @param reproductions Song's reproductions number.
 */
export class Song {
  name: string;
  duration: number;
  gender: string;
  single: boolean;
  reproductions: number;
  constructor(name, duration, gender, single, reproductions) {
    this.name = name;
    this.duration = duration;
    this. gender = gender;
    this.single = single;
    this.reproductions = reproductions;
  }
}

/**
 * Discography.
 * @param name Disc's name string.
 * @param year Year when the disc was launched.
 * @param songs Songs array which conform the disc.
 * @function songs_ Return the song's array.
 * @function songsNumber Return the total number of songs.
 * @function discDuration Return the disc duration.
 * @function reproductions Return the total disc's reproductions.
 */
export class Discography {
  name: string;
  year: number;
  songs: Song[];
  constructor(name, year, songs) {
    this.name = name;
    this.year = year;
    this.songs = songs;
  }
  get songs_() {
    return this.songs;
  }
  songsNumber() {
    return this.songs.length;
  }
  discDuration() {
    let sum = 0;
    for (let i = 0; i < this.songsNumber(); i++) {
      sum += this.songs[i].duration;
    }
    return sum;
  }
  reproductions() {
    let sum = 0;
    for (let i = 0; i < this.songsNumber(); i++) {
      sum += this.songs[i].reproductions;
    }
    return sum;
  }
}

/**
 * Artist.
 * @param name Artist name string.
 * @param listeners Listeners' number.
 * @param discography Discography[] array. 
 */
export class Artist {
  public name: string;
  public listeners: number;
  public discography: Discography[];
  constructor(name, listeners, discography) {
    this.name = name;

/**
 * Function for search an artist in the music library.
 * @param str Artist name to be searched.
 * @returns Artist found.
 */
export function searchArtist(str: string) {
  for (let i = 0; i < MusicLibrary.length; i++) {
    if (MusicLibrary[i].name === str) {
      return MusicLibrary[i];
    }    
  }
}

/**
 * Function for search a discography in the music library.
 * @param str Discography name to be searched.
 * @returns Discography found.
 */
export function searchDiscography(str: string) {
  for (let i = 0; i < MusicLibrary.length; i++) {
    for (let j = 0; j < MusicLibrary[i].discography.length; j++) {
      if (MusicLibrary[i].discography[j].name === str) {
        return MusicLibrary[i].discography[j];
      } 
    }
  }
}

/**
 * Function for search a song in the musin library.
 * @param str Song name to be searched.
 * @returns Song found.
 */
export function searchSong(str: string) {
  for (let i = 0; i < MusicLibrary.length; i++) {
    for (let j = 0; j < MusicLibrary[i].discography.length; j++) {
      for (let z = 0; z < MusicLibrary[i].discography[j].songs.length; z++) {
        if (MusicLibrary[i].discography[j].songs[z].name === str) {
          return MusicLibrary[i].discography[j].songs[z];
        }
      }           
    }
  }
}

Ejercicio 2 - Conecta 4

Este ejercicio consiste en realizar el clásico juego de "Conecta 4", para ello haremos uso de las clases que consideremos necesarias.

El código desarrollado para este juego es el siguiente:

import * as Prompt from 'prompt-sync';
const prompt = Prompt();

type pieces = "x" | "o" | "-";

/**
 * Class game with 2 players.
 * @param player1 First player.
 * @param player2 Second player.
 */
class Game {
  player1: string;
  player2: string;
  constructor(player1, player2) {
    this.player1 = player1;
    this.player2 = player2;
  }
}

/**
 * Class board inherit from class game.
 * @param Board for play "connect 4" 6 rows, 7 cols.
 * @function print print the board.
 * @function isFull check if a column is full.
 * @function insert insert a piece in a column.
 * @function checkWin check if a player has win.
 */
class Board extends Game {
  board: pieces[][] = [["-", "-", "-", "-", "-", "-", "-"],
                       ["-", "-", "-", "-", "-", "-", "-"],
                       ["-", "-", "-", "-", "-", "-", "-"],
                       ["-", "-", "-", "-", "-", "-", "-"],
                       ["-", "-", "-", "-", "-", "-", "-"],
                       ["-", "-", "-", "-", "-", "-", "-"]];
  constructor(player1, player2) {
    super(player1, player2);
  }

  print() { 
    for (let i = 0; i < 6; i++) {
      console.log(`${this.board[i][0]} ${this.board[i][1]} ${this.board[i][2]} ${this.board[i][3]} ${this.board[i][4]} ${this.board[i][5]} ${this.board[i][6]}`);
    }   
  }

  isFull(num: number) {
    for (let i = 0; i < 6; i++) {
      if (this.board[i][num] === "-") {
        return false;
      }      
    }    
    return true;
  }

  insert(pic: pieces, num: number) {
    let i = 5;
    while (true) {
      if (this.isFull(num - 1) === false) {
        if (this.board[i][num - 1] === "-") {
          this.board[i][num - 1] = pic;
          break;
        } else {
          i--;
        }
      } else {
      num = parseInt(prompt('Columna completa. Seleccione otra columna (1-7): '))
      }  
    }
  } 

  checkWin(pic: pieces) {
    let cont = 0;
    // Check horizontal win.
    for (let i = 0; i < 6; i++) {
      for (let j = 0; j < 7; j++) {
        if(this.board[i][j] === pic) {
          cont++;
        } else {
          cont = 0;
        }
        if(cont === 4) {
          return true;
        }
      }      
    }
    cont = 0;
    // Check vertical win.
    for (let i = 0; i < 7; i++) {
      for (let j = 0; j < 6; j++) {
        if(this.board[j][i] === pic) {
          cont++;
        } else {
          cont = 0;
        }
        if(cont === 4) {
          return true;
        }
      }
    }
      cont = 0;
      // Check diagonal win.
        for (let i = 0; i < 6; i++) {
        for (let j = 0; j < i; j++) {
          if (this.board[i-j][j] === pic) {
            cont++;
          } else {
            cont = 0;
          }
            if(cont === 4) {
            return true;
          }
        }        
      }
    return false;
  }
}

Lo primero que hago es declarar el tipo pieces para asegurarnos de que solo se usan dichos caracteres. En mi caso he creado una clase Game que contiene los nombres de los 2 jugadores, esta clase es heredada por Board la cual contiene todas la estructura del juego. El atributo board de tipo pieces[][] es el tablero del juego con todas sus casillas en "-", indicando que están vacías. Primero tenemos el método print(), el cual imprime el estado actual del tablero. El método isFull() comprueba si la columna donde el jugador quiere poner una ficha está llena, ya que en dicho caso deberá de seleccionar otra. Insert() mete la ficha en la columna indicada por el jugador en la fila más baja donde no haya una ficha, simulando la gravedad el juego clásico. Por último tenemos una función checkWin() que comprueba si alguno de los jugadores ha conseguido alineas 4 fichas en vertical, horizontal o diagonal.

La parte que se ejecuta cuando los jugadores están jugando se encuentra en un bucle que finaliza cuando alguno de los jugadores gana. En este bucle se lleva un conteo de los turnos, donde el jugador 1 juega en los turnos impares y el jugador 2 en los pares.
